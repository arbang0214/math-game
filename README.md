# math-game

초등 5~6학년 대상 분수/소수 타임어택 웹 게임

## 컨셉 (2026-07-06 인터뷰 확정)

- **대상**: 초등 고학년 (5~6학년), 불특정 다수 공개 (링크 공유)
- **내용**: 분수/소수 — ① 크기 비교 ② 분수/소수 연산 ③ 약분/통분
- **형식**: 타임어택 아케이드 — 문제마다 제한시간 + 생명(하트) 시스템

## 게임 규칙

```
❤❤❤        점수: 12
██████░░░░ (문제별 타이머)

   3/4  vs  0.7
```

- 문제마다 제한시간이 있고, 레벨이 오를수록 시간이 짧아진다
- 틀리거나 시간 초과 시 하트 1개 감소, 하트를 다 잃으면 게임오버
- 연속 정답 시 콤보 보너스 (점수 배율 증가)
- 최고점 기록 도전 + 전체 리더보드

## 답 입력 방식 (문제 유형별 혼합)

| 문제 유형 | 입력 방식 |
|---|---|
| 크기 비교 (3/4 vs 0.7) | 이지선다 (왼쪽/오른쪽) |
| 분수/소수 연산 | 사지선다 |
| 약분/통분 (동치 분수) | 사지선다 |

## 기술 스택

- 바닐라 HTML/CSS/JS — 빌드 도구 없음, 정적 파일로 완결
- 배포: GitHub Pages (예정)
- 리더보드: 서버 필요 → 후순위 단계 (MVP는 localStorage 최고점만)

## 빌드 단계 (작은 단위, 한 겹씩)

1. ✅ 문제 생성기 — 크기 비교 문제 자동 생성 + 정답 판정 (순수 JS, UI 없음)
2. ✅ 최소 게임 루프 — 문제 표시 → 답 선택 → 정답/오답 → 다음 문제
3. ⬜ 타이머 + 하트 — 문제별 제한시간, 생명, 게임오버 화면
4. ⬜ 점수 + 콤보 — 콤보 배율, localStorage 최고점
5. ⬜ 문제 유형 확장 — 연산(사지선다), 약분/통분 추가
6. ⬜ 레벨/난이도 곡선 — 레벨업마다 시간 단축·문제 어려워짐
7. ⬜ 배포 — GitHub Pages 공개
8. ⬜ 전체 리더보드 — 서버/DB 붙이기 (별도 설계)
