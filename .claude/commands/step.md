빌드 단계 $ARGUMENTS 을(를) 진행한다.

1. README.md의 "빌드 단계"에서 해당 단계의 범위를 확인한다.
   범위가 모호하면 구현 전에 질문한다.
2. CLAUDE.md 컨벤션을 지켜 구현한다 (로직/DOM 분리, rng 주입 등).
3. 순수 로직에는 테스트를 작성한다 (결정적 테스트 + 속성 검사).
4. `npm test`를 실행해 전체 통과를 확인한다.
   출력에 나오는 모든 `결과:` 줄(테스트 파일별 집계)을 보고에 포함한다.
5. README의 해당 단계 체크박스를 ⬜→✅로 갱신하고,
   CLAUDE.md의 "현재 상태" 섹션을 갱신한다. CLAUDE.md의 다른 섹션
   (컨벤션, 작업 방식 등)은 수정하지 않는다.
6. 변경 파일 목록과 남은 미결사항을 요약 보고한다.
