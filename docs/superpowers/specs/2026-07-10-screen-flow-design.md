# 화면 흐름 재설계 + 밸런스 조정 설계 (2026-07-10)

시작/엔딩/리더보드 화면을 신설해 게임의 처음과 끝을 만들고, 게임오버 팝업을
버튼 허브로 개편한다. 함께 목숨·제한시간 밸런스를 완화하고 당근 HUD를 키운다.

## 결정 사항 (브레인스토밍 확정)

- 점수 등록은 **신기록일 때만** (일반 게임오버는 순위 구경만)
- 엔딩 화면에 작은 **[처음으로]** 버튼 (완전한 막다른 길 방지)
- 목숨 3→**5**, 기본 제한시간 10→**12초**, 하한 6→**8초** (레벨당 -0.5초 유지)
- 당근 HUD 1.6→**2rem** 확대

## 화면 5개와 전환 규칙

```
[시작] ──게임 시작──> [플레이] ──게임오버──> [게임오버 팝업]

게임오버 팝업:  다시 시작 → [플레이]
                순위 확인 → [리더보드]
                게임 종료 → [엔딩]
                (신기록) 닉네임+확인 → 등록 → [리더보드]

리더보드:       다시 시작 → [플레이]
                게임 종료 → [엔딩]

엔딩:           처음으로 → [시작]
```

- 오버레이 4개(`#start-screen`, `#gameover`, `#leaderboard-screen`, `#ending-screen`)를
  `hidden` 토글로 전환. 화면 전환은 표시 전용 → game.js에만 둔다
  (game-core.js·problems.js·format.js·leaderboard.js 무변경 — 단, 밸런스 상수만 예외)
- **페이지 접속 시 시작 화면** — [게임 시작] 전에는 게임 상태·타이머가 돌지 않는다
- [다시 시작]은 어느 화면에서든 새 게임 시작(`createGame()`), [게임 종료]는 엔딩 화면으로

## 1. 시작 화면 (`#start-screen`)

- 토끼 `assets/rabbit-idle.png` + 말풍선 **"친구들아, 분수와 소수로 즐거운 게임 해보자!"**
- 큰 **[게임 시작]** 버튼 (기존 `#restart` 스타일 계열)

## 2. 게임오버 팝업 개편 (`#gameover`)

- 유지: 상황 컷(cheer/sad), 제목, 점수·최고점, 신기록 별 연출
- **팝업 안 리더보드 섹션(TOP 10 목록·입력폼) 제거**
- 일반: **[다시 시작] [순위 확인] [게임 종료]**
- 신기록: 닉네임 입력(localStorage `math-game.nickname` 프리필 유지) + **[확인]**
  → 등록 성공 시 리더보드 화면으로 자동 이동. 실패 시 문구 + 재시도.
  아래에 [다시 시작] [게임 종료] 병행(등록 건너뛰기 허용, [순위 확인]은 생략)
- Supabase 미설정(`isConfigured` false): [순위 확인]과 신기록 등록 UI 숨김

## 3. 리더보드 화면 (`#leaderboard-screen`, 신규)

- 🏆 전체 TOP 10 + 상태 문구(불러오는 중/실패/비어 있음) + **[다시 시작] [게임 종료]**
- 진입할 때마다 새로 조회 (기존 `refreshRanking` 세대 가드 로직 이전)

## 4. 엔딩 화면 (`#ending-screen`)

- 토끼 `assets/rabbit-cheer.png` + **"언제든 다시 놀러와!"**
- 구석에 작은 **[처음으로]** 버튼 → 시작 화면

## 5. 밸런스·HUD 조정

- `game-core.js`: `MAX_HEARTS` 3→5, `TIME_LIMIT_MS` 10000→12000,
  `TIME_MIN_MS` 6000→8000 (`TIME_STEP_MS` 500 유지)
- 당근 HUD: `.carrot` 1.6rem→2rem (off 상태 회색 처리 유지).
  당근 5개+레벨·점수 칩이 모바일 폭(28rem)에서 겹치지 않는지 육안 확인
- 순수 로직 테스트: 상수 하드코딩 없이 상수를 참조하는 기존 테스트는 그대로
  통과해야 하고, 값 자체를 단정하는 테스트가 있으면 새 값으로 갱신

- 브라우저 육안: 전체 흐름 순회(시작→플레이→게임오버→순위→다시 시작→신기록
  등록→리더보드→게임 종료→엔딩→처음으로), 타이머가 시작 화면에서 안 도는지,
  당근 5개 레이아웃, 12초 체감
